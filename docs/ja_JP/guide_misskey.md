# Misskeyカスタム絵文字のダウンロード
MisskeyのURLのみが必要で、トークンは不要です。

# Misskeyカスタム絵文字のアップロード
トークンが指定されていない場合、`meta.json`を含むzipファイルが生成されます。このzipファイルは、カスタム絵文字を一括アップロードする際に使用できます。

トークンが指定されている場合、sticker-convertは生成されたzipファイルのアップロードを試みます。
zipファイルのアップロードに失敗した場合、sticker-convertは絵文字を一つずつアップロードしようとします。

# tokenの取得

## 方法1：自動
1. Chromeブラウザをインストールします。
2. MisskeyのURLを入力します。
3. sticker-convert GUIの`生成`ボタンをクリックします。
4. ウィンドウの`tokenを取得`ボタンをクリックし、Misskeyにログインします。

## 方法2：手動
1. ブラウザでMisskeyにログインします。
2. `F12`キーを押して開発者ツールを開きます。
3. `コンソール`に`JSON.parse(localStorage.getItem("account")).token`と入力し、`Enter`キーを押します。
4. 返される結果がトークンです。

# カスタム絵文字の命名規則
カスタム絵文字は、`<カテゴリ>-<名前>.<拡張子>`の形式でダウンロードおよびアップロードされます。

例1：カテゴリ`lol`に属する名前`:cat:`のカスタム絵文字は、`lol-cat.png`となります。
例2：未分類のカスタム絵文字`:dog:`は`dog.png`となります。
