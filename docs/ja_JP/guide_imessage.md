# iMessage スタンプパックの作成方法
iMessage を使用するには、Xcode を使って iMessage にスタンプを読み込むアプリを作成する必要があります。
sticker-convert は Xcode プロジェクトの作成を自動化します。

1. sticker-convert で、圧縮プリセットと iMessage への出力オプションを選択します。
2. sticker-convert で作成された Xcode プロジェクトを開きます。
3. 左上の再生ボタンでスタンプパックをテストできます。これにより、スタンプアプリがビルドされ、エミュレータで起動します。
4. スタンプアプリを Apple デバイスにサイドロードできます（未テスト）。
5. Apple に料金を支払って、誰でも使えるようにスタンプアプリを公開できます。
