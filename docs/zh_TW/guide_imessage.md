# 如何建立 iMessage 貼圖包
iMessage 需要您使用 Xcode 建立一個應用，並將貼圖載入到 iMessage 中。
sticker-convert 可以自動建立 Xcode 專案。

1. 在 sticker-convert 中，選擇壓縮預設並將輸出選項設為 iMessage。
2. 開啟 sticker-convert 所建立的 Xcode 專案。
3. 您可以使用左上角的播放按鈕以測試貼紙包。這將在模擬器中建置並啟動貼紙應用程式供您測試。
4. 您可以將貼紙應用程式側載到您的 Apple 裝置（未經測試）。
5. 您可以向 Apple 付費，並將貼紙應用程式發布供所有人使用。
