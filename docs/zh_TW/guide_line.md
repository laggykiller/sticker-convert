# 下載 Line 貼圖
1. 您可以從 https://store.line.me/stickershop 或 https://store.line.me/emojishop 搜尋貼圖。
2. 複製貼圖包的 URL，並輸入到 sticker-convert 的 URL 欄。

# 下載帶有自訂文字的“訊息貼圖”
1. 輸入您的訊息貼圖 URL，開始轉換。
2. 下載貼圖時會彈出提示訊息，請編輯 line-sticker-text.txt 文件，然後繼續。

# 下載帶有自訂文字的“隨你填貼圖”
此功能需要 Line cookies。

## 方法一：自動轉換
1. 安裝Chrome瀏覽器
2. 在貼圖轉換介面點選`生成`按鈕
3. 在彈出的視窗中點選`取得 cookies`按鈕並登入Line（格式：`key_1=value_1;key_2=value_2`）
4. 輸入您的隨你填貼圖 URL，開始轉換
5. 下載貼圖時會彈出提示訊息，請編輯 line-sticker-text.txt 文件，然後繼續

## 方法二：手動
1. 在瀏覽器中安裝 `Get cookies.txt LOCALLY` 擴充功能
    - Chrome：https://chrome.google.com/webstore/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc
    - Firefox：https://addons.mozilla.org/en-US/firefox/addon/get-cookies-txt-locally/
    - Github頁面：https://github.com/kairi003/Get-cookies.txt-LOCALLY
2. 登入 store.line.me
3. 開啟 `Get cookies.txt LOCALLY` 擴充程序，選擇`Export Format`為`JSON`，然後點擊`Copy`
4. 將複製的內容貼上到sticker-convert中的`Line cookies`

# 上傳 Line 貼圖
您需要手動提交貼圖包進行審核，才能在應用程式中使用。
