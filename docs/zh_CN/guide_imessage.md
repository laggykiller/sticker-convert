# 如何建立 iMessage 貼图包
iMessage 需要您使用 Xcode 建立一个应用，并将貼图载入到 iMessage 中。
sticker-convert 可以自动建立 Xcode 专案。

1. 在 sticker-convert 中，选择压缩预设并将输出选项设为 iMessage。
2. 开启 sticker-convert 所建立的 Xcode 专案。
3. 您可以使用左上角的播放按钮以测试貼纸包。这将在模拟器中建置并启动貼纸应用程式供您测试。
4. 您可以将貼纸应用程式侧载到您的 Apple 装置（未经测试）。
5. 您可以向 Apple 付费，并将貼纸应用程式发布供所有人使用。
